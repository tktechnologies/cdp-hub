# Command: Start Maintenance Chat

1. Pick a prompt from [`.agent/prompts/maintenance/`](../../../.agent/prompts/maintenance/README.md) (or run [../prompts/agent-startup.md](../prompts/agent-startup.md) for a quick bootstrap).
2. Paste the **Prompt** block into a new chat and fill in **My task**.
3. Open the skill listed in the prompt for your specific work.

## Done when

- Agent read the right tier (`scrapers/AGENTS.md` vs platform `AGENTS.md`).
- Production snapshot understood (`MAINTENANCE_CHECKPOINT.md` / `implementation-state.md`).
- Quality gates identified for the change type.
